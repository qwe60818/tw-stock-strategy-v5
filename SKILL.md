# 台股短線策略 v5.0（最終版）

## 策略邏輯

**進場條件**（全部滿足）：
- 股價 > MA20 > MA60 > MA120（嚴格多頭排列）
- MA20 向上（前一日 MA20 < 當日 MA20）
- 量能 > 1.5x（當日成交量 > 5日平均成交量 × 1.5）

**出场条件**（满足任一即卖出）：
- +8% 停利
- -5% 停损
- 10日最大持有

## 交易規則

| 項目 | 內容 |
|------|------|
| 本金 | 50萬 |
| 投入 | 現金90% |
| 持倉 | 可同時持有多檔 |
| 股數 | 優先1000股（整張），不足則100股 |
| 股票池 | 75檔固定 |

## 使用方式

### 1. 執行掃描

```bash
python3 ~/.npm-global/lib/node_modules/openclaw/skills/tw-stock-strategy-v5/scripts/scan.py
```

### 2. 回測

```bash
python3 ~/.npm-global/lib/node_modules/openclaw/skills/tw-stock-strategy-v5/scripts/backtest.py
```

### 3. 訊號輸出格式

```
| 股票 | 股價 | MA20 | MA60 | MA20趨勢 | 量能 |
|------|------|------|------|----------|------|
| 2330 | 1810 | 1867 | 1683 | 向上 | 1.46x |
```

## 注意事項

- 空頭市場（股價 < MA20）自動空手
- 盤整市勝率較低
- 科技股表現優於金融股
